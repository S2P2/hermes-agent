# Overview

The Eko Platform allows organizations to integrate their applications directly into Eko using RESTful APIs. The Eko Client API helps organizations distribute company information, content and documents to employees in a secure channel on single collaboration platform.

### **Highlights**

* **Standard & secure channel to distribute messages, content and document from enterprise to employee.**
* **Automatically create group chats to engage employees in conversations.**
* **Create, update or delete messages and content in Eko collaboration platform using simple API calls. Easily build powerful add-ons to distribute data between business unit and employee**
* **Integrate Chat bot with Eko using Bot Adaptor API**
* **Create and submit workflow through Eko using Workflow API.**
* **Enable SSO for third party web application.**
